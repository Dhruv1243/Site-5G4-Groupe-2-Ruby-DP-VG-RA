Visiter ce site Ruby Download pour installer Ruby et ainsi pouvoir l’exécuter sur votre système. Pour créer un premier programme en Ruby, il suffit d’ouvrir vs code, d’écrire le code et d’enregistrer le fichier avec l’extension .rb, par exemple hello.rb. Pour afficher du texte à l’écran, on utilise la commande puts. Le premier programme le plus simple est puts "Hello World".

```bash
puts "Hello World"

``` 

Ensuite on lance le programme dans le terminal

```bash 

ruby hello.rb 

``` 

Et le output sera :

```bash 

Hello World

```
